# Templates fictícios para Salesforce Marketing Cloud

Esta pasta contém exemplos fictícios de templates HTML utilizados no case study **ETL Pipeline for Marketing Cloud Customer Journeys**.

Os templates demonstram como campos provenientes de uma Data Extension podem ser utilizados em e-mails personalizados por meio de AMPscript.

## Arquivos

- `template_analistas.html`: template com dados do cliente e do consultor responsável.
- `template_gestoras.html`: template com dados do cliente e data de encerramento do contrato.

## Campos utilizados

### Template Analistas

| Campo da Data Extension | Uso no template |
|---|---|
| Email | Destinatário |
| Cliente_Nome | Saudação |
| Contrato_Encerramento | Corpo do e-mail |
| Consultor_Nome | Contato comercial |
| Consultor_Email | Link mailto |
| Consultor_Telefone | Contato telefônico |

### Template Gestoras

| Campo da Data Extension | Uso no template |
|---|---|
| Email | Destinatário |
| Cliente_Nome | Saudação |
| Contrato_Encerramento | Corpo do e-mail |

## Observação

Todos os textos, nomes, links, cores e dados são fictícios. Os arquivos foram criados exclusivamente para fins educacionais e de portfólio.
